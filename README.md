### 🎓 Campus Compass OS

### 🚀 Your Ultimate Fresher Guide for Amrita Amaravati

> One platform. All campus essentials. Built to help freshers navigate university life with confidence.

---

## 🌟 Overview

Starting college can be overwhelming. Important information is often scattered across multiple portals, websites, student groups, and social media pages.

**Campus Compass OS** solves this problem by providing a centralized digital hub for students of **Amrita Vishwa Vidyapeetham, Amaravati**.

Whether you're looking for academic portals, student clubs, campus events, resources, or everyday guidance, Campus Compass OS brings everything together in a clean, modern, and easy-to-use interface.

---

## 🎯 Problem Statement

Freshers often face challenges such as:

❌ Not knowing which portal to use
❌ Difficulty discovering clubs and communities
❌ Lack of awareness about campus activities
❌ Information scattered across multiple platforms
❌ Confusion during the initial weeks on campus

---

## 💡 Solution

Campus Compass OS acts as a digital companion that helps students:

✅ Access essential academic portals instantly

✅ Discover technical, cultural, and sports clubs

✅ Explore campus life and student activities

✅ Learn about university resources

✅ Find answers to frequently asked questions

✅ Adapt to campus life faster and more confidently

## 🚀 Getting Started

### Prerequisites

* Node.js (v18+)
* npm
* Git

### Clone the Repository

```bash
git clone

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

### Build for Production

```bash
npm run build
```
